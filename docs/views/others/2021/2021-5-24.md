TypeScript中interface和type的区别

1. 继承方式

`interface`使用`extends`继承，`type`使用`&`继承

2. interface独有

`interface`可以重复声明，然后里面的变量自动合并，例如：

```ts
interface A {
    name: string;
}

interface A {
    age: number;
}
```

`interface`设置可选属性、只读属性

3. `type`独有

可以声明基本类型别名、联合类型、元组

```ts
type A = {
    name: string;
}

type B = {
    age: number;
}

type C = A | B;
type D = [A, B];
```
