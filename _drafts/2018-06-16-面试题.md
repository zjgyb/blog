Q7: rem和em的区别<br/>
答：`em`继承父级元素的字体大小，相应的倍数。`rem`是基于根元素`<html>`的，也代表倍数。
```
body {
    font-size: 16px;
}

.container {
    font-size: 20px;
}

.container .box1 {
    font-size: 2em; /* 40px */
}

.container .box2 {
    font-size: 2rem; /* 32px */
}
```
